Class: c3.Polar

Defined in: js\c3-polar.coffee
Inherits: c3.Chart

Overview

A chart that uses a polar coordinate system. r is the radial dimension and t is the angular dimension. The top is 0 degrees and the range is expressed in radians so a full circle is 0 to 2Pi. A polar chart can contain multiple layers.
