    Create an area layer for the plot. Setting stack_options will enable the stacking functionality. For normalized data use stack_options.key to define an accessor which provides a key that uniquely describes the stack that each data element should belong to. Here we stack the data based on cause of death.

        layers: [

            interpolate: 'cardinal',
            stack_options: {
                key: (d) => <any>d.cause,
                styles: { 'fill': (stack) => cause_color(stack.key) },
                title: (stack) => stack.key,  // Setup tooltips
