    Structure for US energy data

  • ¶

    Structure for US energy nodes. name for the name of the node. btu represents the energy flow in Quads BTU units.

    interface USEnergyData {
    name: string;
    btu: number;
}
  • ¶

    Structure for the links between US energy nodes. source is the name of the source node for each link. target is the name of the target node for each link. btu represents the energy flow in Quads BTU units.

    interface USEnergyLink {
    source: string;
    target: string;
    btu: number;
}
